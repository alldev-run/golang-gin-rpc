```bash

# 3. 访问服务
# - HTTP API: http://localhost:8080
# - gRPC: localhost:50051
# - JSON-RPC: http://localhost:8081/rpc

# - Consul UI: http://localhost:8500
# - Prometheus: http://localhost:9091
# - Grafana: http://localhost:3000
```
